## 🧬 Active Projects

### 🤖 **Integrity Swarm & CLI**
*   **Description**: A decentralized, message-driven multi-agent system. Specialized agents operate in isolated directories (`workspaces/`) and communicate over an asynchronous SQLite message bus.
*   **Supervisor Heartbeat**: Governed by a heartbeat manager that orchestrates agent wake-cycles, error tracking, and context window pruning.
*   **AST Compiler Engine**: Features an on-the-fly AST analyzer with incremental caching to optimize dynamic agent tool code execution.

### 🎮 **Godot 3D & Graphics Research**
*   **Description**: Building custom shaders (GLSL/GDScript) and designing physics-driven simulations inside Godot 4.x.
*   **Visualizer**: Visualizing system-level telemetry and dynamic data flows as interactive 3D structures.
